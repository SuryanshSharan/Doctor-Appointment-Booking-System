# Doctor Appointment Booking System

A full-stack web application to book doctor appointments online.

## Features

- User authentication
- Book appointments with doctors
- Admin panel for doctors
- Dashboard for users

## Tech Stack

- **Frontend**: React
- **Backend**: Express.js, Node.js
- **Database**: MongoDB
- **Auth**: JWT + Bcrypt
